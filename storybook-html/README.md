# Storybook for HTML

В этом проекте задействуем Storybook для создания пространства для разработки и прототипирования HTML.

## Быстрый старт

Инициализация проекта в автоматическом режиме:

```bash
npx -p @storybook/cli sb init --type html
```

Запускаем!

```bash
npm run storybook
```
